# Outreach & Growth Context: FetchUp

## Strategy: High-Intent Personalization
We don't spam. We find creators who are *currently* losing money and show them the exact proof using FetchUp.

### **Deep-Dive Resources**
- [Outreach Deep-Dive & ICP Definition](./OUTREACH_DEEP_DIVE.md): Detailed breakdown of our product, target audience, and why the previous "Question Open" strategy failed.

## Target Lead Profile (The "Sweet Spot")
- **Niche**: Productivity, Tech, Workflow (e.g., Obsidian, Notion, Apple ecosystem).
- **Size**: 10k to 100k subscribers.
- **Trigger**: "Alternative" videos where multiple software tools are linked.

## The Automation Roadmap
### Module 1: The Lead Scraper (Data Source)
- **Status**: Planned (Design documented in `plans/outreach-automation.md`).
- **Goal**: Weekly pull of 100 leads matching the "Sweet Spot" criteria.
- **Tech**: Python script + YouTube Search API.

### Module 2: Link Analysis (Pending)
- Automating the check of those 100 leads to see who has broken links before manual outreach begins.

### Module 3: Personalization Engine (Pending)
- Using AI to draft the pitch based on the broken links found.
